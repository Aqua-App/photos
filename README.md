## Usage

1. Create db in MySQL shell
2. Create URI env variable
   - *mysql://[username]:[password]@localhost:3306/[database]*
3. npm run seed
4. npm run build
5. npm run start

## Requirements

* Node >=6.13.0
* MySQL

## Dependencies

From within the root directory:

```sh
npm install
```
