## Usage

1. Create db in mysql shell
2. Create URI env variable 
   - *mysql://[username]:[password]@localhost:3306/[database]*
3. npm run seed
4. npm run build
5. npm run start

## Requirements

* Node / NPM
* MySQL
* JS
